# KodNest Premium Build System

A premium SaaS design system built with calm, intentional, and coherent principles. This is not a student project—it's designed to feel like a serious B2C product company.

## Quick Start

```html
<link rel="stylesheet" href="design-system/index.css">
```

## Design Philosophy

- **Calm** - No gradients, no glassmorphism, no neon colors, no animation noise
- **Intentional** - Every spacing, color, and type choice has purpose
- **Coherent** - One mind designed it. No visual drift.
- **Confident** - Large serif headings, generous spacing, clear hierarchy

## Structure

```
design-system/
├── tokens.css      # Design tokens (colors, typography, spacing)
├── base.css        # Global reset and foundational styles
├── components.css  # Reusable component styles
├── layout.css      # Layout structure components
├── index.css       # Main entry point
└── DESIGN_SYSTEM.md # Full documentation

examples/
├── layout-example.html      # Complete layout structure demo
└── components-example.html  # All components showcase
```

## Key Features

- **4-color system** - Maximum colors for consistency
- **8px spacing scale** - Never random spacing
- **Serif headings** - Confident typography
- **Consistent transitions** - 150-200ms, ease-in-out
- **Global layout structure** - Top Bar → Header → Workspace + Panel → Footer
